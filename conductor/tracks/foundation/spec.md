# Track Specification: Project Foundations & Tooling

## Objective

Establish a solid foundation for testing, mocking, and consistent development across all client extensions.

## Requirements

- Introduce a testing framework (Vitest) for the microservice and frontend.
- Setup Mock Service Worker (MSW) for Liferay API mocking.
- Configure standardized linting and formatting.
- Implement a basic health check test to verify the test runner setup.

## Success Criteria

- `npm test` runs successfully in the microservice and frontend.
- MSW is integrated and can intercept a sample request.
- All code follows the established formatting rules.
